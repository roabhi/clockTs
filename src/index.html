    clockTs

    ClockTs is an experiment and playground for testing TypeScript in combination with a few different things I wanted to try both on the front and back-end side of the web.

    On the back-end there is a simple rest API deployed on Heroku (please do not abuse it!) feeding data to the front-end in combination with the luxon library in order to get the (roughly) right time for each city.

    On the front-end you can drag and drop the different clocks you add to the app to rearange the order in which thery are placed. The clocks are persistent between visits via localStorage until you modify the data associated with your own browser config.

    The main focus was to test Typescript and to have some fun with it. I always find that when trying to get an scheduled meeting with somebody from another country I have to go to a website in order to check the offset between me and that person so...this is kind of a personal solution to that. Hope it will help you as well :)
